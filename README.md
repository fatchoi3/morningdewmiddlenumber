# Team Arithmetic Game

A small web game built with React, TypeScript, Vite and Socket.IO.

## 개요
- 호스트가 방을 만들고 팀 수를 설정
- 참가자는 방 코드로 입장하고 팀 선택
- 호스트가 게임 시작을 누르면 팀별/개인별 문제가 배정
- 정답 개수로 팀 승패를 결정!

## 설치
```bash
npm install
```

## 실행
```bash
npm run start
```

웹 클라이언트는 `http://localhost:4173`에서 열리고 서버는 `http://localhost:3001`에서 동작합니다.
