# 숫자 중앙 WebSocket 게임

React를 사용한 실시간 WebSocket 게임입니다. 한 방에 모인 모든 참가자가 숫자를 한 번 입력하고, 모두 입력하거나 호스트가 중지를 누르면 가운데에 가까운 사람이 승리합니다.

## 실행 방법

1. 패키지를 설치합니다.

```bash
npm install
```

2. 서버와 클라이언트를 동시에 실행합니다.

```bash
npm run dev
```

3. 브라우저에서 `http://localhost:5173`을 엽니다.

4. 같은 방 번호를 사용해 다른 탭 또는 다른 브라우저에서 참여합니다.

## 게임 규칙

- 방에 참가한 모든 사용자가 숫자를 한 번 입력합니다.
- 모두 입력하거나 호스트가 `호스트 중지` 버튼을 누르면 게임이 종료됩니다.
- 제출된 숫자들 중 가운데에 가까운 사람이 승리합니다.
- 참가자가 짝수 명일 경우 두 가운데 숫자의 평균값을 목표로 계산합니다.
- 두 명 모두 목표값으로부터 같은 거리라면 두 명 모두 승리합니다.

## 파일 구조

- `src/App.jsx` - React 게임 UI와 WebSocket 상태 관리
- `server/index.js` - WebSocket 서버와 게임 상태 관리
- `src/styles.css` - 간단한 UI 스타일
- `vite.config.js` - Vite 설정
- `package.json` - 실행 스크립트 및 의존성
